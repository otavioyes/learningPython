Vamos construir uma calculadora de gorjetas.

Se a conta foi de $150.00, dividida entre 5 pessoas, com 12% de gorjeta.

Cada pessoa deveria pagar:

`(150.00 / 5) * 1.12 = 33.6`

Após formatar o resultado para 2 casas decimais = `33.60`
